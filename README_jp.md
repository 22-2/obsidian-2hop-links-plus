# Obsidian 2Hop Links Plus リンク解説

このプラグインで表示される各セクションのリンクの意味について解説します。

## 1. Links (順方向リンク)
- **意味**: 現在のノートから直接リンクしているノートです。
- **関係**: `現在のノート` → `表示されるノート`
- **用途**: 今読んでいるノートが参照している情報を素早く確認できます。

## 2. Back Links (逆方向リンク)
- **意味**: 他のノートから現在のノートへリンクしているノートです。
- **関係**: `表示されるノート` → `現在のノート`
- **用途**: このノートがどこから参照されているか、文脈を確認するのに役立ちます。

## 3. 2-Hop Links (2ホップリンク)
- **意味**: 現在のノートと「共通のノート」を介してつながっているノートです。
- **関係**: `現在のノート` → `共通のノート` ← `表示されるノート`
- **詳細**: 直接のリンクはありませんが、同じノートを介して関連している「兄弟」のような関係のノートが表示されます。
- **用途**: 直接リンクを貼っていないけれど、内容的に関連がある可能性が高いノートを発見できます。

## 4. New Links (新規リンク)
- **意味**: 現在のノート内にリンク（`[[ ]]`）はあるが、まだファイルが作成されていないノートです。
- **用途**: まだ存在しないノートへのリンクを可視化し、新しいノートの作成を促します。

## 5. Tag Links (タグリンク)
- **意味**: 現在のノートと同じタグ（`#tag`）を持っているノートです。
- **用途**: 同じカテゴリやテーマで分類されたノートを横断的に探すのに便利です。

## 6. Property Links (プロパティリンク)
- **意味**: 現在のノートと同じプロパティ（Frontmatter）の値を持っているノートです。
- **詳細**: 設定で指定した特定のキー（例: `project`, `author` など）の値が一致するノートが表示されます。
- **用途**: 特定の属性（プロジェクト名や作成者など）に基づいた関連ノートを表示します。
